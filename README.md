# RaiseTech Javaコース 学習記録

エンジニアへのキャリアチェンジを目指し、RaiseTechのJavaコースで学習した内容をまとめたリポジトリです。Javaの基礎から、Spring Bootを用いたWebアプリケーション開発の基礎を習得しました。

## 📋 主要な制作物：ユーザー管理システム（バックエンド）
コースの集大成として、ユーザー情報の管理を行うAPIサーバーを構築しました。

### 実装した主な機能
* **ユーザー情報のCRUD操作:** 登録、一覧表示、詳細参照、更新、削除の基本機能を全て実装。
* **バリデーション:** 入力値の妥当性チェックを行い、適切なエラーレスポンスを返す仕組みを構築。
* **例外処理:** 独自例外クラスを作成し、エラーハンドリングを共通化。
* **単体テスト:** JUnitを用い、各機能が意図通りに動くかテストコードで確認。

## 🛠 修得した技術スタック
* **言語:** Java 17
* **フレームワーク:** Spring Boot
* **データベース:** MySQL, H2 Database
* **ORM:** MyBatis
* **ビルドツール:** Gradle
* **バージョン管理:** GitHub

## 💡 意識したこと
* **保守性の高いコード:** Controller / Service / Mapper の役割を明確に分けるレイヤー設計を意識しました。
* **丁寧なエラーハンドリング:** 利用者が使いやすいAPIを目指し、エラーメッセージの内容にもこだわりました。
