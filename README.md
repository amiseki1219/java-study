# RaiseTech Javaコース 学習記録

エンジニアへのキャリアチェンジを目指し、RaiseTechのJavaコースで学習した内容をまとめたリポジトリです。
Java 21 を使用し、Spring Bootを用いたモダンなWebアプリケーション開発の基礎を習得しました。

## 📋 主要な制作物：ユーザー管理システム（バックエンド）
コースの集大成として、ユーザー情報の管理を行うAPIサーバーを構築しました。

### 実装した主な機能
* **ユーザー情報のCRUD操作:** 登録、一覧表示、詳細参照、更新、削除の基本機能を全て実装。
* **バリデーション:** 入力値の妥当性チェックを行い、適切なエラーレスポンスを返す仕組みを構築。
* **例外処理:** 独自例外クラスを作成し、エラーハンドリングを共通化。
* **単体テスト:** JUnitを用い、各機能が意図通りに動くかテストコードで確認。

## 🛠 修得した技術スタック
* **言語:** Java 21
* **フレームワーク:** Spring Boot
* **データベース:** MySQL, H2 Database
* **ORM:** MyBatis
* **ビルドツール:** Gradle
* **バージョン管理:** GitHub

## 💡 意識したこと
* **保守性の高いコード:** レイヤー設計（Controller / Service / Mapper）を意識し、変更に強い構造にしました。
* **最新仕様の活用:** Java 21の機能を活かし、効率的でクリーンなコーディングを心がけました。
